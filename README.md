## 📘 필독 사항
* 크래프톤 정글 7기 정글러들의 코딩 테스트 대비를 위한 Repository입니다.
* 매일 1문제씩 풀고 일주일동안 푼 문제들을 주말에 PR 합니다
* 논의 후, 팀원들이 일주일 동안 풀었던 문제에 대한 PR을 확인하고 Main 브랜치에 병합합니다.
<br>
